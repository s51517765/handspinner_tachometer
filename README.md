# handspinner_tachometer

## 概要 (Overview)

ハンドスピナー(handspinner,fidget)の回転数を測定します。<br>
光センサ(cds)でハンドスピナーの羽を検出し、FFT(Arduinoのライブラリを使用)で周波数を算出し、回転数を算出します。
